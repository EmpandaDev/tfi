Viabilidad técnica

    Recursos de Hardware

    Especificación de los recursos Hardware

    Tecnologías de Entrada de datos: Mouse, teclado

    Tecnologías de Salida: Pantalla de computadora, notebook y pantalla de celular.

    Tecnologías de Procesamiento: 2 servidores distintos, uno para la página web el cual sería un VPS, en este caso elegiremos Hostwinds.

    Una VPS para alojar un sitio web mediano con las características del nuestro tendría aproximadamente 1000 visitantes por mes y 100 transacciones por mes. Para alojar un sitio web e-commerce como el este, se recomienda que el VPS tenga al menos 2 núcleos de CPU, 4 GB de RAM, 75 GB de almacenamiento y 1 TB de ancho de banda.

    En cuanto al VPS para el servidor interno, las peticiones son mucho menores ya que es para empleados del sistema por lo tanto bastaría con un VPS de 1cpu 2GB de RAM, 30GB de almacenamiento y 1 TB de ancho de banda

    Tecnologías de Almacenamiento: El almacenamiento se realizaría en la nube (VPS) y se eligió junto con el paquete de procesamiento que nos ofrece Hostwinds.

    Esto debido a que, teniendo varias sucursales, asumimos que cada sucursal tiene alrededor de 1000 clientes y cada cliente tiene alrededor de 10 transacciones, entonces el número total de registros sería de alrededor de 170,000. Una base de datos de 1TB puede almacenar hasta 100millones de registros, para nuestra base de datos del servidor interno bastaría con una de 10GB, pero usaremos una de 30GB (mínimo permitido en el plan)

    Para el sitio web también lo elegimos con Hostwinds y el almacenamiento para un sitio web mediano sobra con 1GB, pero el mínimo del plan es de 75GB para 2 nucleos de CPU

    Tecnologías de Comunicaciones: Internet por Fibra ótica, módems y rooters

    Recursos de Software

    Especificación de los recursos de Software

    Sistema operativo: Cualquiera que soporte navegadores como Firefox, Chrome, Opera, Safari. En este caso como las PCs cuentan con Windows instalado, usaremos ese

    Lenguaje de programación: Para la programación web se usará Javascript con librerías React, CSS3 con librerías Boostrap, para el backend se usará el lenguaje GO debido a la posibilidad de manjear concurrencia de procesos

    Antivirus: Se usará Avast Premium Security ya que las coputadoras no cuentan con antivirus y al ser un sistema web, prevenimos malware exterior y otros tipos de amenazas

    Base de Datos: SQL
